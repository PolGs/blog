## Setting Up the Blog

Create a repository on GitHub to hold your blog content with the following directory structure:

```shell
├── about.md
├── posts
  ├── art-of-getting-better.md
  ├── behavior-of-links-created-using-javascript.md
  └── yellow-fade-technique-in-css.md
├── site.json
└── static
    ├── resume.pdf
    └── ebook.pdf
```
